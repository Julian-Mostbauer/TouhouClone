# Touhou-like Game

## Requirements

[Dotnet 9 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/9.0)

## How to play

1. Clone the repository
2. Navigate to the TouhouClone folder
3. Run the game using dotnet

```bash
git clone https://github.com/Julian-Mostbauer/TouhouClone.git
cd TouhouClone/TouhouClone
dotnet run
```

## Controls

- Move: Arrow Keys or WASD
- Shoot: Spacebar
- Quit: Escape
